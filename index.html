How to set up Supabase (free, 5 minutes)

  1. Go to supabase.com and create a free account
  2. Click New project, name it Camp Care, choose region ap-southeast-2
  3. Click SQL Editor and run this SQL:
create table if not exists tasks (
  id text primary key,
  name text, cat text, grp text,
  freq integer, due text, notes text, doc_url text
);
create table if not exists history (
  id uuid default gen_random_uuid() primary key,
  name text, cat text, grp text,
  completed_on text, was_due text,
  outcome text, completion_notes text
);
  1. Go to Project Settings → API
  2. Copy the Project URL and anon public key
  3. Paste them above and click Save & connect
